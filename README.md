### 参与实体(Entities)
具有多个实体,分别是:

- lines
  - ISLs
  - 轨道
  - 路由
- nodes
  - 卫星
  - 地面站
- 其他
  - 地面站信号范围
  - 卫星信号范围
  
其中,ISLs和路由需要多个entities协作\ref,无法将其分开为独立文件
